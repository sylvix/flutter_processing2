# `noStroke()`

```dart
void noStroke()
```

Disables drawing the stroke (outline). If both `noStroke()` and `noFill()` are called, nothing will be drawn to the screen.

## Examples

```dart
size(400, 400);
noStroke();
rect(120, 80, 220, 220);
```
