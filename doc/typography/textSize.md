# `textSize()`

```dart
void textSize(num fontSize)
```

Sets the current font size. This size will be used in all subsequent calls to the `text()` function. Font size is measured in units of pixels.

## Examples

```dart
size(400, 400);
background(0);
fill(255);
textSize(104);
text("WORD", 40, 200);
textSize(56);
text("WORD", 40, 280);
```
